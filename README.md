# Soul Surf Mobile

Aplicativo mobile desenvolvido com Expo e React Native.

## Configuracao de API

Crie um arquivo `.env` baseado em `.env.example`:

```bash
EXPO_PUBLIC_API_URL=http://147.15.58.134:8080
```

Se `EXPO_PUBLIC_API_URL` nao for definido, o app usa automaticamente o endpoint Oracle acima como fallback.
